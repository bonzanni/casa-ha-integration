# Casa HA Integration

Companion integration for the Casa add-on. See [phase 2.4 design spec](https://github.com/bonzanni/casa-ha-app/blob/master/docs/superpowers/specs/2026-04-17-ha-integration-2.4.md) for architecture. Manual E2E checklist lives in §11.2.

## Install

Custom HACS repository. Add `https://github.com/bonzanni/casa-ha-integration` as an Integration repository, install, restart Home Assistant, then add Casa from the integrations page.
